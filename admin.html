🔒 管理者ログイン

編集するには GitHub アクセストークンが必要です。

トークンの作り方（初回のみ）
  1. GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token
  2. Repository access → Only select repositories → countdown-57
  3. Permissions → Contents: Read and write
  4. 発行された github_pat_... を上に貼り付けてログイン

※ トークンはこの端末のブラウザにのみ保存され、サイトには送信されません。
